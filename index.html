    🚀 Novidades do Sistema!

    Confira o que mudou na última atualização

    ✨ Versão 1.5 - Atualizações Recentes

    • Integração com API Federal: Agora é possível consultar benefícios diretos do Portal da Transparência através de CPF ou NIS. O sistema usa Proxy para contornar bloqueios.
    • Busca Local Aprimorada: Busca inteligente por nome com sistema de cache de memória para resultados extremamente rápidos.
    • Visualização Detalhada: Nome completo, CPF formatado e NIS são exibidos no alerta de confirmação de benefício.
    • Assinaturas Dinâmicas: Comprovantes e Liberações agora imprimem automaticamente o nome do usuário/atendente que está operando no momento.
    Dica: Para a API Federal funcionar, lembre-se de configurar sua "Chave API" no menu Admin > Configurações.
